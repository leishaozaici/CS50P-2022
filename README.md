## Why this repository?

Unlike repositories that only upload source code, this project keeps every assignment organized according to the official CS50P curriculum, making it suitable for:

- Reviewing Python fundamentals
- Understanding idiomatic Python solutions
- Practicing problem-solving
- Preparing for interviews
- Following the CS50P learning path
